# TP Gestion de Personnes
Ce projet est un exercice PHP permettant d'ajouter des personnes dans une liste persistante grâce à un fichier JSON.

## Fonctionnalités
- Formulaire d'ajout (Nom, Prénom, Adresse, Téléphone).
- Stockage permanent dans un fichier `pers.json`.
- Affichage dynamique dans un tableau Bootstrap.

## Installation
1. Cloner le projet dans votre dossier `htdocs`.
2. Lancer votre serveur local (XAMPP/WAMP).
3. Accéder à `localhost/votre-dossier/tp2.php`.
